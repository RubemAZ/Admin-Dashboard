# Admin-Dashboard 
  Este projeto é um painel administrativo para diversos tipos de usuários, produtos e serviços.
  
  A aplicação está implantada aqui: https://rubem-adm-dashboard.vercel.app
 
## Stack utilizada:
### Backend: 
  Node v22 <br/>
  TypeScript 5 <br/>
  Prisma v6 <br/>
  Express v4 <br/>
  TS-Jest v29
  
### Frontend: 
  React v18.3 <br/>
  Ant Design 5
